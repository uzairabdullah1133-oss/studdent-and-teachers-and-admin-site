# Apexium Portal

This is a simple web application that provides a login portal for students, teachers, and admins.

## Features

- Login pop-ups for students, teachers, and admins.
- User authentication using data from `data.js`.

## Getting Started

1. Clone the repository.
2. Open `index.html` in your web browser.

## Usage

- Click on the "Login as Student", "Login as Teacher", or "Login as Admin" buttons to open the login pop-up.
- Enter the username and password to log in. The credentials are in the `data.js` file.

## Future Enhancements

- Create dashboard pages for students, teachers, and admins.
- Add more features to the dashboards.
